# BlinkEdge 👁️
### *Empowering Vision Beyond Sight*

> An AI-powered, voice-first resume builder built exclusively for the blind and visually impaired community — fully hands-free, no screen required.

🔗 **Live Demo:** [cerulean-cuchufli-ea0646.netlify.app](https://cerulean-cuchufli-ea0646.netlify.app)  
🏆 **Built at:** STARLET 4.0 — Girls-Only Hackathon, Fort Kochi (Mind Empowered x Lenient Tree)  
🚀 **Submitted to:** Young Innovation Program (YIP) by K-DISC

---

## 💡 Problem Statement

Over 285 million people worldwide are visually impaired. Most resume builders require visual navigation — forms, dropdowns, drag-and-drop interfaces — making them completely inaccessible. Job seekers with visual impairments are left behind before they even get started.

**BlinkEdge changes that.**

---

## 🎯 Solution

BlinkEdge is a fully voice-driven resume builder. A user opens the app, a voice greets them, asks guided questions, collects their information verbally, and generates a professional ATS-friendly resume — all without touching a screen.

---

## ✨ Features

- 🎙️ **Voice Input & Output** — Full Speech-to-Text and Text-to-Speech experience
- 📋 **Guided Resume Creation** — Step-by-step voice prompts for Education, Skills, Experience, Projects
- 📄 **ATS-Friendly Output** — Generates recruiter-ready resume formats
- ♿ **Accessibility-First UI/UX** — Designed with zero visual dependency
- 🔒 **Privacy & Independence** — No data stored; user stays in full control
- 🌐 **No Installation Required** — Fully browser-based, works on any device

---

## 🔄 How It Works

1. User opens the app — a soothing voice greets them
2. Voice prompts guide the user through resume sections
3. User responds verbally — Speech-to-Text captures all input
4. BlinkEdge processes and structures the data
5. A professional ATS-friendly resume is generated
6. User can download or share the final resume
